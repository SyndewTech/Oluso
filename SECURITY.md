# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Oluso, please report it responsibly:

**Email:** security@oluso.io
**PGP Key:** https://oluso.io/pgp-key.txt

### What to Include

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact assessment
- Any proof-of-concept code (if applicable)

### Our Commitment

- Acknowledge receipt within 24 hours
- Provide initial assessment within 72 hours
- Keep you informed of remediation progress
- Credit researchers in security advisories (with permission)
- No legal action against good-faith security research

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
